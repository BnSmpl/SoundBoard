Installation
---

0. Connect to PineBoard

```$ ssh user@soundpi.local```

```$ cd SoundBoard```

1. Create environment

#################################

```
$ python3 -m venv .venv
$ source .venv/bin/activate
$ python3 -m pip install -r requirements.txt
```

#################################

2. Start main program

`$ python3 SoundBoard.py`
